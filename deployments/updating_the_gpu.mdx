To change the GPU and infrastructure settings of your deployment, click the three dots/edit button next to it. 

![Edit comfy deployment](/public/edit_comfyui_deployment.webp)

From there, you can change your deployment name, the GPU model on which it runs, and the maximum number of instances (GPUs) your deployment can use simultaneously. 

Please get in touch if you have special infrastructure requirements. For example, we can adjust the Minimum number of idle instances and the Scaledown window up on requests (team@viewcomfy.com).
