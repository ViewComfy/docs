You can access the Comfy manager via the Comfy interface and use it just like you would on a local installation. 

For example, you can add registered custom nodes from the node manager and unregistered ones via Git. 

![Install custom nodes](/public/install_custom_nodes.webp) 

You can also install most pip dependencies straight from the manager.
