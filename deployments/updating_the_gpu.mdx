To change your deployment's GPU or infrastructure settings, click the three dots/edit button next to it.  

![Edit comfy deployment](/public/edit_comfyui_deployment.webp)

From there, you can change the deployment's name, the GPU model on which it runs, and the maximum number of instances (GPUs) the deployment can use simultaneously. 

Please get in touch if you have special infrastructure requirements. For example, we can adjust the minimum number of idle instances and the scaledown window on requests (team@viewcomfy.com).
